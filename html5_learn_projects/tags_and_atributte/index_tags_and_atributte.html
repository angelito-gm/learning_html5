HTML5

Definición breve de HTML5

HTML; Hypertext Markup Language (Lenguaje de Marcado de Hipertexto) por sus siglas en ingle, es el lenguaje de marcado de la web, es utilizado para la creación de páginas web (estructuración). Se compone de elementos, cada elemento se compone de etique de apertura, atributos(si los hay), contenido y etiqueta de cierre.

Dicho esto, manos a la obra!

Estructura de un documento HTML5:

<!DOCTYPE html> Especifición de la versión html que estaremos usando, en esta caso HTML5.

<html> </html> Etiqueta principal de todo documento html, en ella se inserta todo lo relacioinado con el sitio web, (¡Sin ella es imposible estructurar un documento en la web!)

<head> </head> Etiqueta de vital importancia para nuestro documento html, pues en ella se insertan los metadatos (datos que describen datos), es decir, los metadatos ofrecen información sobre el autor del documento, la codificación de caracteres, el titulo del documento, enlaces a las hojas de estilo, estilos y mucho mas.

<body> </body> Etiqueta representante de todos los elementos que son de caracter visual para el cliente, es decir, lo que nuestro navegador va a mostrar al usurio navegante sin que este tenga que inspeccionar nuestro documento, como sucede con los metadatos.

Estructura de un documento HTML5:
<!DOCTYPE html>

  <head>

  	<title></title>

  	<meta charset="utf-8">

  	<link href="" type="">

  	<style> </style>

  </head> 

  <body>


  </body>

</html>
